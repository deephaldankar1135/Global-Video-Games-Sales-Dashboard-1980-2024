# 🎮 Global Video Game Sales Dashboard (1980–2024)

An interactive Power BI dashboard analyzing **44 years** of video game industry data, covering sales trends, genres, platforms, publishers, and the relationship between critic scores and sales.

## 📊 Key Insights
- Total games analyzed: **225+**
- Global sales tracked in **millions of copies**
- Dominant genres: Shooter, Sports, Role-Playing
- Clear growth spike in the 2010s
- Exploration of whether higher critic scores correlate with higher sales

## 🛠️ Features
- **KPIs**: Total Global Sales, Number of Games, Avg Critic Score, Console Count
- **Trend Analysis**: Global sales over time (line chart)
- **Breakdowns**: Sales by Genre (column chart) + Console Market Share (donut)
- **Rankings**: Top 10 Best-Selling Games
- **Correlation**: Critic Score vs Global Sales (scatter plot with genre coloring)
- Interactive slicers for Year, Genre, and Developer

**Note**: All sales figures are in **millions of units sold** (not monetary revenue).

## 🚀 How to Use
1. Download the `.pbix` file
2. Open it with **Power BI Desktop** (free)
3. Refresh the data if needed (File → Refresh)
4. Interact with slicers and visuals

## 📌 Data Source
Dataset from Kaggle: [Video Game Sales & Industry Data (1980-2024)](https://www.kaggle.com/datasets/bhushandivekar/video-game-sales-and-industry-data-1980-2024)

## 🛠️ Built With
- Power BI Desktop
- Power Query (for data cleaning)
- DAX measures for KPIs
